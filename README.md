# 前言

随着移动互联网的快速发展，线上购物已经成为人们日常生活的一部分。为此，我们开发了一款网上花店微信小程序，为广大用户提供便捷的购花体验。本项目基于Spring Boot框架，结合微信小程序、Java等技术实现。以下是项目的详细介绍。

# 内容介绍

本项目是一款集浏览、选购、支付、订单管理等功能于一体的网上花店微信小程序。用户可以在小程序中查看各种花卉的分类、详情，并将心仪的花卉加入购物车进行结算。同时，后台管理系统方便管理员对商品、订单、用户等进行管理。

# 技术介绍

## 语言：Java

## 使用框架：Spring Springmvc，mybatis，微信小程序

## 前端技术：JS、Vue、css3，Uniapp

## 开发工具：IDEA/Eclipse，Uniapp

## 数据库：MySQL 5.7/8.0

## 数据库管理工具：phpstudy/Navicat

## JDK版本：jdk1.8

## Maven: apache-maven 3.8.1-bin

## 前端环境：Node.Js 12\14\16
